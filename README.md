# QuestRouX  
### A Gamified Campus Orientation Experience for the Roux Institute  

QuestRouX is a mobile-friendly web application that transforms campus orientation into an engaging, game-like experience. Through quests, journaling, photo memories, AI-supported recommendations, and light social features, the app helps new (especially international) students explore the Roux Institute campus with confidence and reflection.

This project focuses on **HCI-centered design**, **AI-supported decision guidance**, and **playful interaction models** inspired by apps like Duolingo.

##  Core Features

### ** 1. Quest System (RPG-style)**
- **Available**, **Completed**, and **Locked** quests  
- Multiple quest types:
  - Location quests (QR/code-based)
  - Journal quests
  - Photo quests  
  - Final ‚ÄúYearbook Entry‚Äù quest  
- XP-based progression  
- Each quest has rewards, hints, and campus relevance

---

### ** 2. Tracked Quest System**
Users can track **one quest at a time**, which:
- Appears **pinned** and visually highlighted
- Shows a glowing effect
- Highlights its marker in the Map view

---

### ** 3. Map View**
A simplified interactive map showing:
- Points of interest
- Related quests
- Tracked quest marker highlighted
- GPS mock-integration (reads your device location when possible)

---

### ** 4. Journal System**
Journal quests let users reflect on:
- Meeting an advisor  
- First week experiences  
- Navigating campus challenges  

Entries appear in a dedicated Journal tab.

---

### ** 5. Photo Memories & Yearbook**
Photo quests allow users to capture:
- Favorite campus spots
- Selfies or visual reflections

Photos are stored in a memory grid.  
The final quest unlocks a **gold-framed Yearbook entry**.
